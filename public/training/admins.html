06 / Security Architecture

Invisible Security Layers

The platform implements a state-of-the-art **Row-Level Security Structure** through Laravel's global database query scopes, ensuring strict data compartmentalization.

🔒 Branch Manager Scopes: Automatically restricts managers to files belonging only to clients inside their branch.

🔒 Employee Scopes: Restricts standard accountants to view only their assigned portfolios.

🔒 Client Portal Isolation: Isolates clients to only view their own profile and verified ledgers.

07 / Audit Protocol

Ledger Audits & Sign-offs

When employees submit monthly financial ledgers, they enter a **Submitted** draft state. Admins review the entries and sign off on verified records.

👁️ Verification Locking: Approved ledgers are locked automatically. The client can only view **Verified** statements in their portal.

🔑 Unlock-to-Edit Switch: If corrections are needed, Admins toggle the secure switch to unlock the entries and make adjustments.
